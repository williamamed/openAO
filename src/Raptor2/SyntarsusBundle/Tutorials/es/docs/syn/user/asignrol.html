Asignando un rol
Una vez que tengamos el usuario adicionado debemos asignarle
un rol, debido a que las especificaciones del RBAC establecen
que el usuario tomará sus privilegios de acuerdo al rol que
ocupe.
Para la asignación de rol selecccionamos el usuario en la
lista y pulsamos sobre el botón asignar rol donde escojemos
de la lista de roles disponibles el que le asignaremos
al usuario.
