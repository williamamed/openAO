Bienvenido a la Guía interactiva para el módulo de Seguridad Syntarsus,
a través del módulo de seguridad podrás administrar todos los usuarios que tendrán
permisos para acceder a las funcionalidades protegidas de tu aplicación web.

¿Que debo conocer?
Esencialmente Syntarsus implementa una extensión del modelo de control de acceso
RBAC que a su vez está basado en estructuras, significa que los usuarios siempre
serán asignados a estructuras definidas permitiendote un mayor alcance en la
implementación de sistemas con estructuras organizacionales.
