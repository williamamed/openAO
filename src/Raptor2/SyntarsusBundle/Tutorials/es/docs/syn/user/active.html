Activación de usuarios
Una vez que tenemos insertado un usuario debemos activarlo
ya que por defecto aparece en estado inactivo y por lo tanto
no tendrá acceso al sistema.

Para esto buscamos en la lista de usuarios el usuario deseado
y pulsamos en la casilla de estado que aparece en la parte derecha,
deberá desplegarse un combobox con las opciones de activación y
desactivación del usuario. Una vez hecho esto ya el usuario queda
habilitado para el acceso o no.
