Bueno ahora veremos la parte más importante

El administrador de privilegios te permitirá registrar todos
los permisos definidos para tu aplicación.

Los permisos están organizados en un árbol que además de organizarlos
servirán como una estructura para el retorno del menú de usuario, lo
que significa que mediante un servicio de seguridad puede obtenerse
un menú de acceso a estos privilegios para el usuario autenticado, solo
se retornarían los privilegios a los que este usuario posee acceso.
